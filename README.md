# トランシーバ記録システム

トランシーバ（無線機）の音声をリアルタイムで文字起こしし、話者を自動識別してExcelに記録するデスクトップアプリです。文化祭・イベントのスタッフ交信ログ作成を想定して設計されています。

## 機能

- **自動文字起こし** — mlx-whisper（whisper-large-v3）による日本語音声認識
- **話者識別** — Resemblyzerによる声紋照合（コサイン類似度）
- **2種類の検出モード**
  - PTTモード：有線接続したイヤホンジャックの信号エネルギーでPTT押下を検出
  - VADモード：Silero VADによる音声区間検出（マイク・デバッグ用）
- **リアルタイム信号レベルメーター** — PTT送信中の視覚フィードバック
- **Excelへの自動書き込み** — 時刻・話者・発言内容を既存ファイルに追記
- **直前の記録を修正** — 誤認識をその場で編集して再出力

## スクリーンショット

| DB作成タブ | 記録・出力タブ |
|---|---|
| 話者ごとに声サンプルを録音してDBに保存 | 発言を検出するたびに自動でExcelへ書き込み |

## 動作環境

- macOS（Apple Silicon 推奨 — mlx-whisperがApple SiliconのNeural Engineを使用）
- Python 3.10 以上

## セットアップ

```bash
# 依存ライブラリのインストール
pip install -r requirements.txt
```

### requirements.txt

```
torch
mlx-whisper
sounddevice
resemblyzer
numpy
xlwings
openpyxl
silero-vad
```

> **Note:** `mlx-whisper` はApple Silicon（M1/M2/M3）専用です。Intel Macでは通常の `openai-whisper` への差し替えが必要です。

## 起動方法

```bash
python main.py
```

## 使い方

### 1. 声紋DBの作成（DB作成タブ）

登録したい人数分、以下の手順を繰り返します。

1. 「名前」欄に話者名を入力
2. 「録音開始」→ トランシーバで発言 → 「録音停止」
3. サンプルを複数録音して精度を上げる（任意）
4. 「DBに保存」でvoiceprint.npzに書き込み

### 2. 交信の記録（記録・出力タブ）

1. 「出力先」でExcelファイル（.xlsx）を選択
2. 検出方式を選択
   - **PTT信号（有線接続）** ― 有線接続したトランシーバのジャック音声を使う場合
   - **VAD（マイク/デバッグ）** ― PCのマイクで集音する場合
3. 「記録開始」をクリック
4. 発言が検出されるたびに自動で文字起こし・話者識別・Excel書き込みが行われる
5. 誤認識があれば「直前の記録」欄で修正して「直前を修正・再出力」

### Excelの出力形式

| 時刻 | 話者 | 発言内容 | タグ | 要約 |
|------|------|----------|------|------|
| 10:32:15 | 山田 | 正門の搬入完了しました。どうぞ | | |

## ファイル構成

```
transceiver_logger/
├── main.py          # エントリーポイント
├── ui.py            # GUIアプリ本体（Tkinter）
├── audio_capture.py # 音声入力ストリーム管理
├── vad.py           # PTT検出器 / Silero VAD
├── transcriber.py   # mlx-whisperによる文字起こし
├── speaker_id.py    # Resemblyzerによる話者識別
├── db_manager.py    # 声紋DB（.npz）の読み書き
├── excel_writer.py  # xlwingsによるExcel書き込み
└── requirements.txt
```

## 設定値の目安

| パラメータ | デフォルト | 説明 |
|---|---|---|
| PTT閾値 | 0.002 | RMSエネルギーの閾値。有線接続ノイズに合わせて調整 |
| 話者識別閾値 | 0.75 | コサイン類似度。低いと誤認識増、高いと「不明」増 |
| 最小発言長 | 0.5秒 | これより短い音声セグメントは無視 |

## 注意事項

- `voiceprint.npz`（声紋DB）はgit管理対象外です。各環境で個別に作成してください。
- Excel書き込みには`xlwings`を使用しているため、Excelアプリが起動していても書き込み可能ですが、ファイルが開かれていると競合する場合があります。
- PTTモードは有線接続（イヤホンジャック経由）を前提としています。Bluetooth接続では動作しません。
